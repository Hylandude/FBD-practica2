Requerimientos para ejecutar el programa:
Java SDK 13 o superior - https://www.oracle.com/java/technologies/javase-jdk13-downloads.html
Apache Ant             - https://ant.apache.org/manual/install.html

Para ejecutar el programa:
1.- En la terminal navegar a la carpeta que contiene el archivo build.xml
2.- Ejecutar el comando "ant run". Esto crea las carpetas requeridas, compila, crea documentacion de java y ejecuta el programa.
